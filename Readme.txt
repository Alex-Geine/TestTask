компиляция проводилась в операционной системе Windows 10 c помощью Sygwin и компилятора gcc
Процесс компиляции:
1) Необходимо поместить папку с исходниками в директорию Sygwin (я поместил в папку home).
2) В терминале открыть эту папку cd (путь\к\папке).
3) Ввести в териминале g++ -std=c++20 GamingClub.cpp main.cpp -o task
4) Запустить программу ./task.exe test1.txt
